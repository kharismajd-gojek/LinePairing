#  Line Library

## Problem Statement
A library to calculate length of a line in a cartesian system

## Pre-Requisite
1. [JDK 11](https://www.oracle.com/java/technologies/downloads/#java11)
2. [Gradle 6.9.1](https://gradle.org/releases/)

## Running Test
```bash
./gradlew test
```

## Building the Library
```bash
./gradlew build
````
